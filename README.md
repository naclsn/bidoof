collection of lib-ish things to play around with things

```console
$ vi some.c
$ make build/some
$ build/some
```

- base: buffer manipulation, common things (like fs files)
- archives: zip
- checks: crc32, adler32
- compressions: inflate
- encodings: base64, utf8
- encryptions: rot, vigenere
- images: png
- views: nothing
